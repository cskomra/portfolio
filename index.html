Bug Evaders Arcade Game

This game is much like Frogger in that, as a player, you are trying to cross a sidewalk full of bugs in order to get to the other side. Making it to the other side gets you to the next level. Each one of the three levels has a unique set of bugs that travels at different speeds, making the game more challenging. Click here to try it out.

Neighborhood Map

This ia a single-page web application built using the Knockout framework. It displays a Google Map of an area with various points of interest. Users can search all included landmarks. When a marker is selected, additional information about the selected landmark is presented from either FourSquare or Wikipedia. Click here to try it out.
